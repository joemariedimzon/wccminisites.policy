wccminisites.policy Installation
--------------------------------

To install wccminisites.policy using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``wccminisites.policy`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        wccminisites.policy

* Re-run buildout, e.g. with:

    $ ./bin/buildout
